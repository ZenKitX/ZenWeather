# 禅意天气 🌤️

一个融合中国传统文化的天气应用，用诗意的方式感受天气变化。

## ✨ 特色功能

### 🎨 禅意美学

- 水墨风格配色（竹绿、宣纸白、梅粉等）
- 浅色/深色主题自动适配
- 优雅的动画效果
- 简洁清爽的界面设计

### 📖 文化底蕴

- **60+首古诗词** - 根据天气智能匹配相关诗词
- **24节气系统** - 识别当前节气并展示节气诗词
- **四季主题** - 春夏秋冬自动切换配色方案
- **天气动画** - 雨、雪、云、晴等生动动画效果

### 🌍 实用功能

- **实时天气** - 获取当前位置的实时天气数据
- **多城市管理** - 添加、切换、管理多个城市
- **天气详情** - 24小时预报和7天预报
- **详细信息** - 湿度、风速、能见度、气压等
- **下拉刷新** - 快速更新天气数据

### ⚙️ 个性化设置

- 深色模式切换
- 温度单位切换（℃/℉）
- 语言选择（中文/英文）
- 清除缓存功能

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd zen_weather
```

### 2. 安装依赖

```bash
flutter pub get
```

### 3. 配置 API Key

打开 `lib/utils/constants.dart`，替换 API Key：

```dart
static const String weatherApiKey = '你的API密钥';
```

获取 API Key：

1. 访问 [WeatherAPI.com](https://www.weatherapi.com/)
2. 注册账号并登录
3. 复制 Dashboard 中的 API Key

### 4. 运行应用

```bash
flutter run
```

详细说明请查看 [快速启动指南.md](快速启动指南.md)
