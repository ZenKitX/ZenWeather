## 💡 技术实现

### 1. 城市管理

```dart
// 搜索城市
Future<void> searchCities(String query) async {
  final results = await _weatherService.searchCities(query);
  searchResults.value = results.map((location) => 
    CityModel.fromLocation(location)
  ).toList();
}

// 添加城市
Future<void> addCity(CityModel city) async {
  final weather = await _weatherService.getWeatherByCity(city.name);
  cities.add(city.copyWithWeather(weather));
  await _saveCities();
}

// 删除城市
Future<void> removeCity(int index) async {
  cities.removeAt(index);
  await _saveCities();
}
```

### 2. 设置管理

```dart
// 切换主题
void toggleTheme() {
  isLightTheme.value = !isLightTheme.value;
  StorageService.setThemeMode(isLightTheme.value);
  Get.changeTheme(
    isLightTheme.value 
        ? ZenTheme.getThemeData(isLight: true)
        : ZenTheme.getThemeData(isLight: false),
  );
}

// 切换温度单位
void toggleTemperatureUnit() {
  isCelsius.value = !isCelsius.value;
}
```

### 3. 路由导航

```dart
// 跳转到城市管理
final result = await Get.toNamed(AppRoutes.cities);

  controller.refreshWeather();

```

## 🚀 使用流程

### 添加城市
1. 点击首页位置 → 进入城市管理
2. 点击右上角 [+] → 进入搜索页面
3. 输入城市名称 → 显示搜索结果
4. 点击城市 → 添加成功
5. 返回城市列表 → 查看新城市

### 切换城市
1. 进入城市管理页面
2. 点击任意城市卡片
3. 自动返回首页
4. 显示选中城市天气

### 修改设置
1. 点击首页设置图标
2. 进入设置页面
3. 切换相应选项
4. 设置自动保存
