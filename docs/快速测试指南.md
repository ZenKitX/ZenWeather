# 快速测试指南

## 🚀 5分钟快速测试

### 步骤 1: 配置 API Key（2分钟）

1. 访问 https://www.weatherapi.com/signup.aspx
2. 使用邮箱快速注册（无需验证）
3. 登录后，在 Dashboard 复制 API Key
4. 打开 `lib/utils/constants.dart`
5. 替换 API Key：
```dart
static const String weatherApiKey = '你复制的API密钥';
```

### 步骤 2: 运行应用（1分钟）

```bash
cd zen_weather
flutter run
```

### 步骤 3: 授予权限（30秒）

应用启动后会请求定位权限，点击"允许"。

### 步骤 4: 查看效果（1分钟）

应用会自动：
1. 获取你的当前位置
2. 显示当地天气
3. 匹配相应的诗词
4. 展示天气详情

## 🐛 常见问题排查

### 问题 1: 无法获取天气数据
**检查项**：
- [ ] API Key 是否正确配置
- [ ] 网络连接是否正常
- [ ] 查看控制台错误日志

**解决方案**：
```bash
# 查看详细日志
flutter run --verbose
```

### 问题 2: 定位失败
**检查项**：
- [ ] 是否授予定位权限
- [ ] 设备定位服务是否开启
- [ ] 模拟器是否设置了位置

**解决方案**：
- Android: 设置 → 应用 → 禅意天气 → 权限 → 位置
- iOS: 设置 → 隐私 → 定位服务 → 禅意天气

### 问题 3: 字体加载慢
**原因**：首次使用 Google Fonts 需要下载

**解决方案**：
- 等待字体下载完成
- 或使用本地字体

### 问题 4: 骨架屏不显示
**检查项**：
- [ ] 是否导入了 shimmer 包
- [ ] 主题颜色是否正确

**解决方案**：
```bash
flutter clean
flutter pub get
flutter run
```
